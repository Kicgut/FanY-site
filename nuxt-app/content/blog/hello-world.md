## 欢迎来到我的个人网站

这是一个集成了多种功能的个人站点，主要包括：

- 📝 **技术博客** — 记录学习心得与实践经验
- 📸 **照片管理** — 支持远程上传、本地存储、相册管理
- 🤖 **AI 内容生产** — 通过 Hermes Agent 辅助内容创作

## 技术架构

```
本地服务器（无公网）                ECS（公网入口）
├── 原图/私密内容存储               ├── Nuxt 3 网站
├── Hermes AI 主实例               ├── Nginx 反向代理
├── Skills API (port 9800)         ├── frps 服务端
└── frpc 客户端 ───隧道───→        └── SQLite 数据库
```

### 核心技术栈

| 技术 | 用途 |
|------|------|
| Nuxt 3 | 全栈框架，SSR/SSG |
| Vue 3 + TypeScript | 前端开发 |
| Prisma + SQLite | 数据库 ORM |
| Element Plus | UI 组件库 |
| Nuxt Content | Markdown 内容管理 |
| frp | 内网穿透隧道 |
| Docker | 容器化部署 |

## 设计原则

1. **本地优先** — 原图、私密内容存储在本地服务器
2. **ECS 做入口** — 公网访问、轻量展示
3. **AI 不能自动发布** — 所有 AI 生成内容需人工审核
4. **分层隔离** — 公开/朋友/私人内容严格隔离
