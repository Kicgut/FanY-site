# 28 小阶段详细规划（教学 + 实战）

> 每个小阶段 = 一个独立可交付的里程碑 + 对应的知识教学
> 预计总周期：18 天，每小阶段 0.5-1 天

---

## 阶段 ① 基础搭建（Day 1-2）

### 1.1 项目初始化与 Nuxt 3 核心概念
**任务目标：**
- 安装 Node.js、pnpm，用 `npx nuxi init` 创建 Nuxt 3 项目
- 理解 Nuxt 3 目录结构（pages/、components/、public/、nuxt.config.ts）
- 运行开发服务器，浏览器打开看到 Hello World

**知识教学 — "Nuxt 3 是什么"：**
- SSR vs CSR vs SSG 三种渲染模式的区别（画图讲解）
- Nuxt 3 = Vue 3 + 自动路由 + 服务端渲染框架
- `pages/` 目录自动变成路由（文件即路由）
- `nuxt.config.ts` 是项目的"控制中心"
- **人格提炼：** 你是一个"全栈学徒"，正在从零搭建自己的数字空间

**交付物：** 一个能跑起来的 Nuxt 3 项目，浏览器看到 Hello World

---

### 1.2 Vue 3 基础与首页设计
**任务目标：**
- 学习 Vue 3 核心语法：模板语法、响应式数据（ref/reactive）
- 编写首页 `index.vue`：个人介绍、导航栏、基础布局
- 使用 Element Plus 安装并引入基础组件（按钮、卡片、布局）

**知识教学 — "Vue 3 的核心思想"：**
- 响应式数据：为什么改变变量，页面自动更新？（Vue 的 Proxy 机制）
- 模板语法：`{{ }}`、`v-bind`、`v-on` 是什么
- 组件化思维：页面 = 搭积木，每个 .vue 文件 = 一块积木
- `<template>` + `<script setup>` + `<style>` 三段式结构
- **人格提炼：** 你是一个"积木搭建师"，学会用组件搭建页面

**交付物：** 一个有导航栏和基本内容的首页

---

### 1.3 布局系统与全局样式
**任务目标：**
- 创建 `layouts/default.vue`（全局布局：顶部导航 + 内容区 + 底部）
- 学习 Nuxt 的 `<NuxtLayout>` 和 `<NuxtPage>` 系统
- 配置全局 CSS 变量（主题色、字体、间距）
- 实现响应式布局（手机/平板/桌面适配）

**知识教学 — "布局是一种设计模式"：**
- 为什么需要布局？（避免每个页面重复写导航和页脚）
- CSS 变量（`--primary-color`）是什么，为什么比直接写颜色好
- 响应式设计：`@media` 查询、Flexbox/Grid 布局基础
- 移动端优先（mobile-first）设计理念
- **人格提炼：** 你是一个"空间设计师"，理解页面布局的骨架

**交付物：** 完整的全局布局，导航可切换页面，手机端自适应

---

### 1.4 路由系统与页面导航
**任务目标：**
- 创建 `/about`、`/blog`（占位）、`/portfolio`（占位）页面
- 使用 `<NuxtLink>` 实现 SPA 无刷新导航
- 学习动态路由参数（`[id].vue`）
- 添加页面过渡动画（`<NuxtPage :transition>`)

**知识教学 — "前端路由的魔法"：**
- 传统网站 vs SPA 的导航区别（整页刷新 vs 局部替换）
- 文件系统路由：`pages/blog/index.vue` → `/blog`
- 动态路由：`pages/blog/[id].vue` → `/blog/123`（`useRoute()` 读参数）
- `<NuxtLink>` vs `<a>` 的区别（预加载、无刷新）
- **人格提炼：** 你是一个"城市规划师"，设计网站的路线图

**交付物：** 多页面可互相跳转，带过渡动画，理解路由系统

### 6.2 Nginx 反向代理配置
**任务目标：**
- 安装 Nginx（容器化或直接安装）
- 配置反向代理：`your-ip:80` → Nuxt 应用

**知识教学 — "Nginx 是互联网的交通枢纽"：**

- 缓存策略：`expires`、`Cache-Control` 头部
- Gzip 压缩：减少传输体积，提升加载速度
