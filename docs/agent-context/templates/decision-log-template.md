# Decision Log Template

每次做出重要架构或权限决策时，在 `docs/decision-log/` 新建一份记录。

```markdown
# ADR-YYYYMMDD-标题

## 状态

proposed / accepted / rejected / superseded

## 背景

为什么需要这个决策？

## 决策

最终选择是什么？

## 备选方案

- 方案 A
- 方案 B

## 影响

- 对安全的影响
- 对实现复杂度的影响
- 对用户体验的影响
- 对运维的影响

## 后续动作

- [ ] action 1
- [ ] action 2
```
