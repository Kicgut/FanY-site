# 学习笔记索引

学习笔记按阶段递增维护，用于保留实施过程、技术选择和复盘，不作为当前架构或运行事实来源。

- `v1/`：Nuxt、页面、内容、数据库和后台基础。
- `v2/`：权限、照片、AI、内容流水线、Skills 与运维阶段。
- `v3/`：生产加固、Jobs、存储一致性和文档治理。
- `deployment/`：Docker 与虚拟机部署学习材料。

需要当前事实时，回到 [`../architecture/current-architecture.md`](../architecture/current-architecture.md)；需要运行步骤时，回到 [`../operations/`](../operations/)。
