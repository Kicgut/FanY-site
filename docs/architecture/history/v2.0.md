# 架构历史 v2.0

## 总体架构

第二阶段将站点扩展为具备权限、照片工作流、内容流水线和外部集成能力的 Nuxt/Nitro 单体。ECS 继续承载生产容器与持久化卷，Ubuntu 通过 FRP 提供内部照片与 Skills 服务。

## 关键变化

- 引入用户、分组、资源归属、审核与审计等权限边界。
- 建立照片导入、缩略图同步、原图回流和可见性控制链路。
- 引入 AI Gateway、Skills 只读索引和内容流水线。
- 将部署、备份、恢复、监控和生产验证写成可执行手册。

后续已完成的安全增强、AI 会话管理、Jobs 与存储运维进入当前 `v3.0.0` 架构。
